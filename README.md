# Course Materials for "Introduction to Data Science and Machine Learning"

Note: everything in this repo is *currently* under construction. The script is not finished yet. Cross-references may be broken. You'll find TODOs. 

## Script

This script is meant to be a comprehensive content and learning materials reference. Two versions:

- raw markdown script with entry point: [script main index](docs/index.md) (the source).
- a html version that is available here: [GitHub Pages](https://fgnussbaum.github.io/ds-ml-intro-mdscript/) (compiled using Python `mkdocs`).

## How and where to read the script

You have several options:

- using the html-compiled version in browser: [GitHub Pages](https://fgnussbaum.github.io/ds-ml-intro-mdscript/),
- using raw markdown in Browser: GitHub renders markdown automatically,
- after download (e.g., `git clone` repo, which will also allow you to pull updates):
  - in VSCode IDE: `Ctrl+Shift+V` on Windows shows a rendered markdown-preview 
  - or put the files into any markdown-compatible note-taking app (e.g., Obsidian)
